API standard Java utilizzate (non terze parti):
   • javax.sound.midi (Java SE)
   • javax.swing (Java SE)
   Questi moduli sono forniti dall’ambiente Java e non sono inclusi in questo archivio.

Come verificare le versioni effettive installate in Processing:
   Processing → Sketch → Import Library… → Add Library… (oppure controlla la cartella
   {sketchbook}/libraries/<nome-libreria>/library/ per leggere il numero di versione).
